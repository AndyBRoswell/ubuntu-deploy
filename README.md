# ubuntu-deploy
Automatically install the essentials on Ubuntu.

# Features

## Shell Aliases

```bash
alias cd.='cd ..'
alias cd..='cd ../..'
alias cd...='cd ../../..'
alias cd....='cd ../../../..'
```

- Consolas
